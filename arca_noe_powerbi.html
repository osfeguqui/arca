Integracion Power BI - Guia

1Alojar el HTML

Sube arca_noe_powerbi.html a SharePoint, Azure Blob o GitHub Pages (HTTPS).

2Instalar visual HTML Content de Orbis en AppSource

3Tabla de capas DAX

CapasArca = DATATABLE("ID",STRING,"Nombre",STRING,"Elementos",INTEGER,{ {"hull","Casco Wood_OSB",14},{"keel","Quilla Fondo",4}, {"ribs","Costillas Support",24},{"outerSupport","Outer Support",14}, {"floors","Cubiertas Floor",3},{"topBottom","topBottom",2}, {"roof","Techo dos aguas",7},{"window","Tzohar Glass",2}, {"door","Puerta Hinge Screw",8},{"fence","Fencing Lattice",10}, {"ground","Agua Rock",4},{"ref","Escala humana",6} })

4Medida HTML con postMessage

HTML_Arca = VAR _ids = CONCATENATEX(VALUES(CapasArca[ID]), CHAR(34) & CapasArca[ID] & CHAR(34),",") RETURN "<script>(function(){ var f=document.getElementById('af'); if(f)f.contentWindow.postMessage({type:'SET_LAYERS',layers:[" & _ids & "]},'*'); })();</script> <iframe id='af' src='https://TU-URL/arca_noe_powerbi.html' width='100%' height='520px' frameborder='0'/>"

5Slicer de capas

Segmentador con CapasArca[Nombre]. Al seleccionar capas, el modelo 3D actualiza en tiempo real.

Mensajes postMessage soportados

// Power BI a visor { type: "SET_LAYERS", layers: ["hull","ribs","floors"] } { type: "SET_VIEW", view: "iso|front|side|top|stern|under" } { type: "HIGHLIGHT", layer: "roof" } { type: "RESET" } // Visor a Power BI (al clic) { type: "LAYER_CLICK", layer: "ribs", name: "Costillas Support" }
